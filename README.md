# 机器作曲

使用遗传算法自动生成音乐旋律的Python实现。

## 快速开始

### 运行程序
```bash
python main.py
```

### 程序输出

1. 生成若干个ABC记谱法文件（`melody_1.abc`, `melody_2.abc`, `melody_3.abc`）
2. 尝试生成MIDI文件（需要安装midiutil）

### 播放生成的旋律

1. 访问 https://abcjs.net/abcjs-editor.html
2. 将生成的ABC文件内容复制到编辑器
3. 点击播放按钮聆听

## 其它

- ABC记谱法标准：http://abcnotation.com/
